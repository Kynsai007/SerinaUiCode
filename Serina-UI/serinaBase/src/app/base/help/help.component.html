Help guide for Quick-upload
Use Quick Upload for Faster Processing
  • In quick upload, you submit the invoices and submit it to the Serina system.
  • If there is no issue on the invoice, it will directly go into Approvals(if selected) or to the ERP.
  • If there are any issues, it would appear under Exceptions Screen.
Entity Selection
  • Only certain Entity access is given to each user
  • Please make sure that only the correct Entity is selected. Any wrong selection against the invoice copy uploaded, would go into exceptions due to Entity mismatch
Vendor Selection
  • Select the relevant vendor from the dropdown.
  • Green Color Coded Vendor means the vendor is onboarded into Serina and model is available.
  • Red Color indicates - vendor invoice templates not onboarded to Serina.
  • If you have an invoice where the vendor is in red category, please contact Serina Support Team to onboard the vendor.
  • Please share with them the 5 invoice samples for onboarding. Once they confirm, you can start uploading the invoices. Now the color code would change to green
Invoice Type
  • There are 4 Invoice Type in Serina for AGI
Single PO
  • This option is selected when you have an invoice against one particular PO only, can be used it again for another invoice for same PO.
  • When this option is selected, selecting the PO number is mandatory.
Multiple PO
  • This option is selected when against once invoice, you have multiple PO to book against.
  • Once this option is selected, a pop up screen appears.
  • In this screen, user has to select the PO and GRN combination for each line. Once done, click add and submit.
Non-PO
  • This option is selected when the invoice has to be posted as journal voucher.
LCM
  • Landed Cost Module is selected when there is allocation required for certain voyage/transport across entities.
  • Once you click add and upload, you should select the LCM allocation from the exception screen. Once done, it will move forward to the next steps.
Invoice Attachment
  • Only attach the Invoice Copies. Multiple pages of same invoice number is allowed. Don't attach PO and GRN copies.
  • Any other documents other than invoice copies would lead to low OCR accuracy and longer upload times.
Supporting Document
  • PO, GRN copies or any other relevant documents against this invoice can be uploaded against "Supporting Document"
  • Don't upload the Invoice copy in this category
Approvals
  • Select "Pre-Approved" if invoice is not going to follow an approval process
    • or
  • Click Add approvers, based on the setup, you can select the approvers and click Okay.
  • Two level approvers would mean the invoice will be going to 2 Approvers
Add
  • Click Add to the invoice to the queue. This would mean that you can add more invoice to the queue before it can be sent for bulk upload.
Upload
  • You can directly upload if you have only 1 invoice to be uploaded.
  • If you have multiple invoice copies, then click Add first. Add the 2nd copy. Then click Upload to upload all together.
